# kokoto

*Kokoto* is a brand-new platform for efficient and flexible cooperation.

## Features

Check out our [Introduction video]() and [paper]().

- Beautiful theme 
- Tag classification & search
- Inline-comment
- History archiving
- Extensible editor
- ...

## Installation

```bash
git clone https://github.com/hatamake/kokoto.git
cd kokoto
npm install
```

## Usage

```bash
npm start
```
